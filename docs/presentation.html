What actually moved the score

  • Disabled reasoning tokens in every category
  • Moved maths and logic into one real-Python batch
  • Forced computed-variable interpolation in submit()
  • Merged all other categories into one direct batch
  • Kept terse, category-specific answer recipes
  • Matched the organiser-style reference judge locally
prompt + output + reasoning = ranked tokens 3,013 + 1,813 + 0 = 4,826
The leaderboard target was ~4,300. The current cloud-only result is 4,826 at 19/19; further reduction needs a reliable zero-token local offload, not weaker answers.

Production auto mode

  • Keep ids, questions, constraints, code, maths, and logic as text
  • Render only quoted sources >=2,000 characters (~500 tokens)
  • Label every image immediately before its image_url wire part
  • Fall back to text on threshold, render, or mode mismatch
  • Keep aggressive tasks/dense/full modes for research
Rejected experiment: two ~450-character images saved only 13 prompt tokens and scored 18/19. Auto now stays text on the real set, preserving 19/19.
